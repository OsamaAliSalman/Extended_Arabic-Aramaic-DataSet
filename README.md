# Extended_Arabic-Aramaic-DataSet

## Introduction
The taxa are the followings scripts or script variants: Old_Aramaic Imperial_Aramaic Hebrew Parthian Nabataean Middle_Persian Palmyrene Hatran Syriac_Estrangela Syriac_Nestorian Armazian Khwarazmian Sogdian Arabic_African Arabic_early Arabic_Kufic Arabic_Mauritanian Arabic_modern and Arabic_Naskh.

The examined taxa features are extracted based on the collections of similar features of the script variants. These collections are called Similarity Features Groups (SFGs), 
and the purpose of their construction is to avoid the random coincidence of evolutionary unrelated but similar features (Hosszú, 2017: 182; Hosszú, 2023).
